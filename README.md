# CustomDungeons v2.0

Элитный плагин на кастомные данжи для Minecraft 1.16.5.

## Новое в версии 2.0
- **Уникальные структуры**: Теперь плагин поддерживает генерацию "блочных" структур прямо из кода/конфигов (Block-based generation).
- **Массовая генерация**: Команда `/cdungeon random` спавнит по 10 уникальных данжей в каждом мире.
- **Отчет о координатах**: После массовой генерации все координаты выводятся одним сообщением в чат для администрации.
- **Оптимизация**: Код переписан по стандартам крупных проектов.

## Возможности
- Кастомная генерация при загрузке новых чанков.
- Поддержка схематик (WorldEdit API).
- Поддержка SQL и MySQL для хранения данных.
- HEX и RGB цвета в сообщениях.
- Поддержка Base64 голов и кастомных предметов.
- GUI для настройки лута и шансов.
- Автор: `aniemsao`.

## Команды
- `/cdungeon gui` - Открыть редактор лута.
- `/cdungeon spawn` - Заспавнить схематику на вашей позиции.
- `/cdungeon random` - Массовая генерация уникальных данжей по всем мирам.
